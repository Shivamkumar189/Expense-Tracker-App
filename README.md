# Expense Tracker App using Data Science

## 📌 Overview

This project is a simple attempt to understand how data can help in tracking and analyzing personal expenses.

Instead of just storing numbers, the goal here is to make sense of the data — to see where money is being spent, how spending changes over time, and what patterns can be observed.

Since I didn’t have access to real financial data, I created a synthetic dataset and built the complete analysis pipeline on top of it.

---

## 🎯 Problem Statement

Most people track expenses manually or not at all. Even when data is available, it’s not always easy to understand spending behavior.

This project tries to solve that by:

* Organizing expense data
* Categorizing spending
* Visualizing trends
* Generating simple insights

---

## 💡 What This Project Does

* Tracks expenses based on categories
* Analyzes spending patterns
* Shows monthly income vs expense trends
* Visualizes data using charts
* Provides basic insights into financial behavior

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/your-username/Expense-Tracker-App.git
cd Expense-Tracker-App
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate        (Windows)
source venv/bin/activate     (Mac/Linux)
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the project

```
python main.py
```

### (Optional) Run dashboard

```
streamlit run app.py
```
